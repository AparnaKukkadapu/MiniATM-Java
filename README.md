Concepts Practiced

---

How to Run

1. Ensure Java JDK (8 or above) is installed.
2. Save the file as:
   MiniAtmProject.java
3. Open terminal in the file directory.
4. Compile the program:
   javac MiniAtmProject.java
5. Run the program:
   java MiniAtmProject
